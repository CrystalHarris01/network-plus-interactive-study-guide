# Network+ Interactive Study Guide

An interactive CompTIA Network+ study guide built with HTML, CSS, and JavaScript.

This project turns Network+ review into a more engaging learning experience with flashcards, quizzes, port drills, XP, streak tracking, and command practice.

## Live Demo

GitHub Pages URL after deployment:

```text
https://crystalharris01.github.io/network-plus-interactive-study-guide/
```

## Features

- Interactive flashcards
- Network+ quiz mode
- Ports and protocols drill
- XP and streak tracking
- Network command practice
- Responsive cyber-themed design
- No backend required

## Network+ Topics Covered

- OSI Model
- TCP/IP fundamentals
- DNS
- DHCP
- Common ports and protocols
- Subnetting basics
- Wireless basics
- Network troubleshooting
- Network security fundamentals
- Linux and network commands

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages
